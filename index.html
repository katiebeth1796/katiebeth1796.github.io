Web Design Basics

Setting Up an HTML Document

  1. Type "<!DOCTYPE html>" at beginning of document
  2. Create html and head tags
  3. Create title within head section(shows up as tab on site)
  4. Link style sheet by typing <link href='style.css' rel='stylesheet'>
  5. Insert body tags
  6. Create title using h1 tags at beginning of body
  7. Create divisions
  8. Begin adding content
